Frequently Asked Questions

Jawaban atas pertanyaan yang sering diajukan pengguna.

Secara garis besar, siklus penggunaan aplikasi terdiri dari 4 tahapan utama:

  • Buat Jadwal: Notulis mendaftarkan rapat baru untuk mendapatkan Link Absensi & QR Code.
  • Pelaksanaan & Absensi: Saat rapat berlangsung, peserta melakukan absensi digital (QR/Link).
  • Pencatatan: Notulis mencatat poin pembahasan pada menu "Tulis Notulen" (Matriks Masalah & Dokumentasi).
  • Finalisasi: Notulis melakukan Generate Laporan Resmi (Doc) dan mengunduh Daftar Hadir (Excel) pada menu "Arsip Notulen" untuk didistribusikan.

Aplikasi E-Notulen dirancang secara inklusif untuk seluruh Civitas Hospitalia RSUD dr. R. Koesma Tuban. Sistem ini dapat digunakan oleh jajaran Direksi, Manajemen, Kepala Instalasi/Ruangan, Komite Medis/Keperawatan, hingga staf administrasi unit untuk keperluan pengelolaan rapat dinas, diskusi klinis, maupun pertemuan internal unit kerja guna mendukung budaya kerja paperless dan tertib administrasi.

Sistem E-Notulen menerapkan prinsip One Meeting One Folder. Saat Anda membuat jadwal rapat, sistem otomatis membuat folder di Google Drive dengan format nama YYYY-MM-DD [Nama Agenda]. Semua file (undangan, foto, excel, doc) akan tersimpan rapi di sana.

Sistem E-Notulen menerapkan kebijakan Privasi Data User. Setiap notulis (User) hanya memiliki akses penuh terhadap data rapat yang mereka buat sendiri. Anda tidak dapat melihat, mengedit, atau menghapus arsip notulen milik rekan kerja lain, begitu pula sebaliknya. Akses monitoring seluruh data hanya dimiliki oleh pengguna dengan hak akses ADMIN.

Login & Akses

Panduan masuk ke dalam sistem.

1

Akses URL

Buka aplikasi E-Notulen melalui browser (Chrome/Edge/Safari). Halaman login akan menampilkan logo RSUD dan tombol Google.

2

Google One Tap

Klik tombol "Sign in with Google" atau tunggu popup muncul di pojok kanan atas. Pilih akun email instansi atau email pribadi yang terdaftar.

Catatan: Jika akun Anda baru pertama kali login, status Anda adalah "USER". Hubungi Admin untuk upgrade akses.

Membuat Jadwal Rapat

Langkah awal memulai notulensi.

Fitur ini digunakan untuk mendaftarkan rapat baru ke dalam sistem agar link absensi dapat dibuat.

  • Unit Pelaksana: Pilih unit kerja penyelenggara rapat. Jika tidak ada, pilih "Lainnya" dan ketik manual.
  • Upload Undangan: Anda bisa mengunggah foto/PDF surat undangan sebagai arsip digital.
  • Otomatisasi Folder: Sistem otomatis membuat folder di Google Drive dengan format YYYY-MM-DD [Nama Agenda] untuk menyimpan semua file terkait rapat ini.

Sistem Absensi Digital

QR Code dan Tanda Tangan Online.

Untuk Peserta

  • Scan QR Code yang diberikan notulis.
  • Isi Nama Lengkap & Unit.
  • Tanda tangan pada layar HP.
  • Klik "Konfirmasi Kehadiran".

Untuk Notulis

  • Buka menu "Arsip Notulen".
  • Klik ikon QR Code .
  • Tampilkan di layar proyektor.
  • Klik ikon Users untuk cek siapa yang sudah hadir.

Menulis Notulen

Fitur Matriks & Dokumentasi.

Sistem ini mendukung pencatatan notulen berbasis Matriks Masalah yang terintegrasi dengan Google Spreadsheet.

Tombol "Buat Notulen Spreadsheet"

Tombol hijau ini akan otomatis membuat file Google Sheet baru dari template standar RSUD. Anda bisa mengisi kolom: No, Uraian Masalah, Tindak Lanjut, PJ, dan Target Waktu di sana.

Dokumentasi

Anda dapat mengunggah foto kegiatan atau melampirkan link materi rapat. File akan otomatis tersimpan di folder Google Drive rapat tersebut.

Laporan & Output

Generate dokumen otomatis.

Laporan Resmi (Google Doc)

Klik ikon di daftar arsip. Sistem akan menggabungkan data rapat, daftar hadir, dan isi matriks spreadsheet menjadi satu dokumen laporan resmi yang siap cetak/TTD.

Daftar Hadir (Excel)

Buka popup kehadiran, lalu klik tombol "Download Excel". File Excel akan berisi daftar hadir lengkap dengan Gambar Tanda Tangan yang tertempel rapi di kolomnya.

Anda juga bisa membagikan hasil rapat langsung ke WhatsApp menggunakan tombol Broadcast.

Panel Admin

Fitur khusus pengelolaan sistem.

Hanya user dengan role ADMIN yang dapat mengakses menu ini.
